# Release Note - News Scraper (v1.1.0)

본 릴리즈는 국내/외 경제 및 증시 뉴스를 봇 차단 없이 안전하고 신속하게 수집하고, 거래대금 상위 특징주들의 상승 요인(뉴스)을 마크다운 보고서에 클릭 가능한 원본 링크(hyperlink) 형태로 시각화하는 고도화 패치입니다.

---

## [v1.1.0] - 2026-05-27

### 📢 주요 변경 사항 (Key Highlights)

1. **로컬 구글 뉴스 디코딩 체계 전환 (`googlenewsdecoder` 도입)**
   - 기존의 빈번한 웹 요청 방식의 구글 뉴스 URL 디코딩 로직을 로컬 라이브러리(`googlenewsdecoder`)를 이용한 오프라인 복호화 체계로 100% 전환했습니다.
   - 이를 통해 봇 감지 차단을 완벽히 우회하고 수집 처리 속도를 대폭 단축했습니다.

2. **네이버 모바일 뉴스 검색 스크래핑 체계 구축**
   - 봇 감지 정책이 까다로운 PC 네이버 검색 대신 모바일 검색(`m.search.naver.com`) 및 모바일 헤더를 적용하여 원천적으로 차단을 우회했습니다.
   - 네이버의 잦은 HTML CSS 클래스명 변경(난독화)에 영향받지 않도록, 동적 클래스를 완전 배제하고 **[텍스트-URL 견고한 매칭(Robust Filtering) 알고리즘]**을 도입해 5개의 특징주 기사를 항상 정확하게 수집하도록 개선했습니다.

3. **특징주 뉴스 원본 링크(Hyperlink) 렌더링 지원**
   - 한국 거래대금 상위 100개 종목 테이블의 하단에 출력되는 특징주 상승 이유(뉴스 리스트)를 단순 텍스트가 아닌, 제목과 원본 링크가 결합된 클릭 가능한 마크다운 링크 포맷(`[기사 제목](URL)`)으로 정렬하여 사용자 편의성을 높였습니다.

4. **네이버 OpenAPI 의존성 및 요약 오버헤드 완벽 제거**
   - 뉴스 수집 오버헤드를 유발하고 API 할당량 제한을 겪던 네이버 OpenAPI 요약 hooks를 삭제하고, RSS 본문 추출 및 다중 폴백 요약 알고리즘을 최적화했습니다.

5. **매크로 뉴스 '마감시황' 단독 섹션 렌더링 지원**
   - 연합인포맥스 RSS 등을 통해 수집한 매크로 기사들을 카테고리 기사 목록과 섞지 않고, 보고서의 상단 요약 근처 영역에 **`## 📰 마감시황`**이라는 독자적인 2단계(H2) 마크다운 헤더 아래에 별도로 출력해 가독성을 끌어올렸습니다.

### 🛠️ 세부 변경 및 버그 수정 내역

- **Fix**: `scraper_kr.py` 내부에서 BeautifulSoup의 로컬 섀도우 임포트로 인해 간헐적으로 발생하던 Python `UnboundLocalError` 스코핑 버그를 완전 해결했습니다.
- **Added**: `config/settings.py`에 확장 및 유지보수가 용이한 국내 매크로 뉴스 RSS 피드(`macro_rss_feeds`) 설정 구조를 추가하여 향후 새로운 채널(예: 연합뉴스 등)을 설정만으로 확장할 수 있게 했습니다.
- **QA**: `tests/test_kr_news_changes.py` 테스트 슈트를 작성 및 수행하여 로컬 디코더, 연합인포맥스 RSS, 네이버 모바일 수집기 등의 안정성과 무결성을 성공적으로 검증했습니다.

### 🚀 실행 방법 (Execution Guide)

- **한국 증시 시황 및 뉴스 수집 파이프라인 실행**:
  ```bash
  python src/main_kr.py
  ```
- **수집 보고서 출력 경로**:
  - `data/output/kr_economy_news.md` (자동 생성 및 실시간 동기화)

- **테스트 슈트 수행**:
  ```bash
  python tests/test_kr_news_changes.py
  ```
